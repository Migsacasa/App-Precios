# App-Precios

## Overview
App-Precios is a TypeScript-based application designed to manage and display pricing information. This project serves as a template for building scalable applications with a focus on type safety and clarity.

## Project Structure
```
App-Precios
├── src
│   ├── app.ts          # Entry point of the application
│   └── types
│       └── index.ts    # Type definitions and interfaces
├── package.json         # npm configuration file
├── tsconfig.json        # TypeScript configuration file
└── README.md            # Project documentation
```

## Installation
To get started with the App-Precios project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/Migsacasa/App-Precios.git
   ```

2. Navigate to the project directory:
   ```
   cd App-Precios
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To run the application, use the following command:
```
npm start
```

This will compile the TypeScript files and start the application.

### Port configuration
- Default API port: 3001
- Override with an environment variable:
   ```
   PORT=3000 npm start
   ```
